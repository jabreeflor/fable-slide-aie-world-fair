Quick Hit · Fable

Fable came back mid-conference

Export-controlled off every cloud, un-banned overnight into Day 3. The schedule was rewritten and the first "Field Guide to Fable" opened the morning.

  • Models are "grown, not designed": capability arrives in spikes, and the harness always lags the model.
  • Anthropic cut 80% of Claude Code's system prompt. The new class wants context, not constraints.
  • Working with Fable = finding your unknowns: blind-spot passes, interviews, quizzes before you merge.

Quick Hit · Personal Claws

Every harness becomes a claw

A claw is a coding harness that grew initiative. It doesn't wait to be prompted.

  • What gets added: heartbeats, daemons, memory.
  • Claude Tag's unlock: Claude wakes itself up to do work.
  • Seen on stage: a founder texting his agent from Telegram to spin up cloud workspaces.
Sam Bhagwat (Mastra): "Every harness will become a claw"
Quick Hit · Multiplayer

Agents went multiplayer

The unit of work is no longer one engineer, one agent, one terminal.

  • Humans around agents: shared live workspaces. "Orchestras, not factories."
  • Agents pulling humans in: Claude Tag wakes itself; Cursor's agents page the researchers.
  • Agents with agents: "a big trend in the next six months."
Quick Hit · Memory

Memory is personalization, and it can't be outsourced

  • Every top consumer AI built its own memory. The surprise: neither ChatGPT nor Claude ever used RAG for it.
  • The converged pattern: a running profile, visible and editable, plus search tools. Its size is a GPU-budget decision.
  • Continual learning is already here, outside the weights: memory vaults, pinned threads, agents that "act and write back."
"Self-improvement and memory," from the main stage
Quick Hit · Security

Security got its own room

The first-ever dedicated Security track, because agents made prompt injection "a bigger problem than SQL injection ever was."

  • The lethal trifecta: private data + untrusted content + tools. An agent with all three is unsafe while reasoning.
  • The consensus: treat the agent as an adversary. MicroVMs, network allow-lists, its own identity, ephemeral runtimes.
  • The elegant bit: credential injection. Secrets usable by the agent, never accessible to it.
Threat modeling, live in the Security track
Quick Hit · Agent Readiness

When agents fail, it's the environment, not the model

  • Unprepared codebases degrade 3–16× more after agent adoption. Same models, same speed.
  • In prepared pipelines, AI-written PRs already match or beat human quality.
  • Readiness is measurable: an eight-dimension hygiene check of your codebase.
Greptile's revert-rate data on the big screen
Quick Hit · Local & Open Models

The frontier is only six months ahead of open

  • GLM-5.2 lands "between Opus 4.7 and 4.8" on the hardest agentic benchmarks.
  • MiniMax M3: top-five, natively multimodal, 1M context, open weights.
  • The subtext is political: with frontier releases government-gated, capability you can hold cannot be revoked.
Day 2 main stage: the GLM-5.2 and MiniMax keynotes
Quick Hit · Valuemaxxing

Tokenmaxxing is out. Valuemaxxing is in.

OpenAI's coinage for the year's economic turn: measure what agents accomplish, not what they burn.

  • The bottleneck moved: tokens → compute → attention. "Unlike tokens, I can't simply add more of it."
  • Right-size the model: "using Opus to summarize a Slack thread is a bad choice."
  • One 16.5-hour mission: 779M tokens, and 745M were cache reads. The cache-hit ratio is the whole cost story.
"Building's easier. Generating value is still hard."

Deep Dive · Loops

Stop prompting agents. Design loops that prompt them.

  • Blind loops produce 40,000-line PRs nobody reads.
  • Control loops make small, verified changes: cost and risk bounded by design.
  • Factory's missions spend ~40% of runtime on validators that never write code.
Agentic control loops (HumanLayer, Day 2)
Deep Dive · Evals + Harness

"Vibe-based engineering is dead"

  • Evals have a half-life. When every model scores ~90%, retire it. Instruction-following got 10× better in one year.
  • Models now attack the evals: caught mining git history and web forks for answers. Keep evals private and held out.
  • The missing benchmark is maintainability. Until it's trainable: we're stuck reading the code.
Vending-Bench: "(mis)behavior found"
Deep Dive · Context Engineering

Agents get bad after ~100k tokens

  • Enterprise reality: ~330 tools, ~47k tokens of schemas, and 5% of sessions use one.
  • The fix: progressive disclosure, a compact index up front, detail loaded on demand.
  • The live debate: curate for control, or discover: "I don't filter anything."
Progressive disclosure, in one line

A menu says "Tiramisu, $9," not the recipe. The agent's tool index works the same way: one line per tool, and the full 2,000-token schema loads only when it actually reaches for it.
