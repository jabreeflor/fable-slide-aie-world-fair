# AI Engineer World's Fair 2026 · Field Report

**Live deck: https://jabreeflor.github.io/fable-slide-aie-world-fair/**

A 22-slide HTML presentation recapping the AI Engineer World's Fair 2026 (June 29 – July 2, Moscone West, San Francisco): Fable's mid-conference return, computer use, personal claws, multiplayer agents, memory, security, agent readiness, open models, valuemaxxing, and deep dives on loops, evals, and context engineering.

Design follows the Anthropic slide style: serif type on warm cream, white cards, hand-drawn SVG diagrams, and photos and clips shot at the conference.

## Controls

| Key | Action |
|---|---|
| `→` `↓` `Space` `PageDown` / click | Next slide |
| `←` `↑` `PageUp` / click left edge | Previous slide |
| `F` | Toggle fullscreen |
| `Home` / `End` | First / last slide |

Deep-link to any slide with a hash, e.g. [`#6`](https://jabreeflor.github.io/fable-slide-aie-world-fair/#6) for the Fable slide.

## Run locally

No build step — open `index.html` in a browser.
